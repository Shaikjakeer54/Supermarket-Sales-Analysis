# Supermarket Sales Analysis Dashboard

## Tools Used
- Power BI
- SQL
- Excel
- DAX

## Project Overview
This project analyzes supermarket sales data to identify sales trends, top products, and customer purchasing patterns.

## Key Insights
- Top 5 products generated most revenue.
- Sales performance analyzed by region.
- Customer purchase trends identified.
- Interactive dashboard built using Power BI.

## Skills Demonstrated
- Data Cleaning
- SQL Queries
- Data Visualization
- Business Reporting
